1 - Para ser possível executar o programa é necessário possuir um interpretador de python instalado
2 - Ter as seguintes bibliotecas instaladas no interpretador
	i - xlsxwriter (pip install xlsxwriter)
	ii - tqdm (pip install tqdm)
3 - Executar, ou através da linha de comandos, IDE ou clicando com o botão direito selecionar a opção abrir como e escolher o interpretador de python
4 - Incerir o número de simulações a realizar e primir enter
